# The Tiny Times 🗞️

A daily kid-friendly newspaper generator that creates a printable, two-page newspaper for young readers. Each issue includes age-appropriate news stories, local events, weather, fun facts, and a coloring page on the back.

## Features

- **AI-generated news** — Pulls real headlines and rewrites them for kids (ages 4–8) using Claude
- **Local events** — Fetches neighborhood events from community calendars
- **Weather** — Current conditions with a kid-friendly prompt
- **Coloring page** — Randomly selected from a library of uploaded illustrations, printed as the back page
- **Print-first design** — Optimized for clean, ink-efficient printing on a single double-sided sheet

## How It Works

1. Configure your child's name, city, and neighborhood
2. Click **Generate** to fetch today's news, events, and weather
3. Click **Print** to print a two-page newspaper (front and back)

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Lovable Cloud (database, storage, edge functions)
- Claude API for news generation
- Lovable AI for coloring page generation
